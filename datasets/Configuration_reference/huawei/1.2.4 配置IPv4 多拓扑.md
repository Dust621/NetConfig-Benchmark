# 1.2.4 配置IPv4 多拓扑

在IPv4网络中，通过配置多拓扑，系统可以更加合理地分配网络资源。

## 应用环境

在传统意义上的IP网络里，仅存在一个单播拓扑，转发层面也只有一份单播转发表。所有去往同一个目的IP地址的业务共享完全相同的逐跳转发行为。这样使得端到端的各种类型的业务（比如语音，数据等）共享相同的物理链路，可能导致某些链路非常拥挤，而某些链路带宽空闲。一个自然的解决方法是在同一个物理网络上为不同的业务规划出不同的逻辑拓扑，称之为多拓扑。

## 前置任务

在配置IPv4多拓扑之前，需完成以下任务：

- 配置接口的链路层协议参数和IPv4地址，使接口的链路协议状态为Up。

## 操作步骤

1. 执行命令 `system-view`，进入系统视图。

2. 执行命令 `ip topology topology-name`，创建拓扑。

3. 执行命令 `interface interface-type interface-name`，进入指定接口视图。

4. 执行命令 `ip topology topology-name enable`，在接口视图下使能多拓扑。

5. 执行命令 `commit`，提交配置。

## 检查配置结果

完成配置后，可以通过 `display ip topology` 命令来检查配置结果。
